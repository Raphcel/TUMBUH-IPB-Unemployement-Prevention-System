Deskripsi Pekerjaan

Sebagai Data Analyst Intern, kamu akan berkesempatan untuk bekerja dengan tim data dalam mengolah, menganalisis, dan menyajikan data untuk mendukung pengambilan keputusan bisnis.

  • Mengumpulkan, membersihkan, dan menganalisis data
  • Membuat visualisasi data dan dashboard
  • Mendukung tim dalam penyusunan laporan dan insight
  • Berkolaborasi dengan berbagai tim untuk memahami kebutuhan data

Kualifikasi

  • Mahasiswa aktif D3/S1 semua jurusan (diutamakan Statistika, Matematika, Sistem Informasi, atau terkait)
  • Menguasai Excel, SQL, dan tools analisis data (Python/R/Tableau menjadi nilai tambah)
  • Memiliki ketertarikan pada data dan pemecahan masalah
  • Teliti, komunikatif, dan mampu bekerja dalam tim

Keuntungan

Uang Saku Kompetitif Pengalaman Kerja Nyata Mentoring Sertifikat Fleksibel Kesempatan Full-time
